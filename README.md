# getFandomFounder

Gets the information about the founder of a [Fandom](https://fandom.com) wiki.

2. Press `Shift` + `CTRL` + `J` (on Windows/Linux) or `Option` + `âŒ˜` + `J` (on macOS) to open the Chrome console.
